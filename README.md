# Xbox Clone

Proyecto frontend que replica la interfaz de la página oficial de **Xbox**, desarrollado con **React** y estilos en **SCSS**.  
El objetivo es practicar la construcción de interfaces modernas, organización de componentes y despliegue en GitHub Pages.

---

## Características
- Interfaz inspirada en la web oficial de Xbox.
- Componentes reutilizables en React.
- Estilos con **SCSS** para mayor flexibilidad.
- Responsive design para desktop y mobile.
- Despliegue en **GitHub Pages**.

---

## Tecnologías utilizadas
- **React 18**  
- **React Scripts** (CRA)  
- **SCSS/SASS**  
- **JavaScript (ES6+)**  
- **GitHub Pages** para despliegue
